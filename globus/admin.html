Configuração inicial

Configure o token de acesso ao GitHub uma vez. Só o dono do site precisa fazer isso.

Crie um token em github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Permissão necessária: Contents: Read and write no repositório jhonatanrlp/remo.

O token fica salvo só no seu navegador (localStorage). Nunca vai para o código.
